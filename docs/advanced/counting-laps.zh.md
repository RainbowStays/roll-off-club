# 计算圈数

## 概述

滚轮行动中，一“圈”是指橙轮或蓝轮完整地转动360°。在一次完整的五分钟平冠中，橙轮和蓝轮都会转12圈。

计算圈数对于路线选择很有帮助，尤其是版型难度较高的时候。

## 圈数对应的时间

| 圈数 | 开始时间 | 持续时间 |
| ----- | ---------- | -------- |
| 1     | 0:00       | 0:47     |
| 2     | 0:47       | 0:32     |
| 3     | 1:16       | 0:29     |
| 4     | 1:41       | 0:25     |
| 5     | 2:05       | 0:24     |
| 6     | 2:28       | 0:23     |
| 7     | 2:51       | 0:23     |
| 8     | 3:12       | 0:22     |
| 9     | 3:33       | 0:21     |
| 10    | 3:54       | 0:21     |
| 11    | 4:15       | 0:21     |
| 12    | 4:36       | 0:21     |

## 实用技巧

如果你仔细听游戏内的背景音乐，你可以在第7圈开始时察觉到背景音乐刚好开始重复第二遍。

从第9圈开始，粘液会上涨至最高高度，这意味着最后四圈通常是难度最高的。
